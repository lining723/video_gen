# Quick Start: 前端页面框架优化

**Feature**: `20260505-frontend-layout-sidebar`
**Date**: 2026-05-05

## 前置条件

1. 前端服务运行在 `http://127.0.0.1:3100`
2. 至少有一个已创建的项目

## 快速开始

### 1. 查看固定菜单栏

1. 打开 `http://127.0.0.1:3100`
2. 点击任意项目进入项目页面
3. 观察左侧菜单栏固定在视口左侧
4. 滚动页面内容，验证菜单栏不随内容滚动

### 2. 收起菜单栏

1. 在项目页面，点击菜单栏底部的收起按钮 (←)
2. 观察菜单栏宽度变窄，仅显示图标
3. 观察主内容区域宽度自动扩展

### 3. 展开菜单栏

1. 在收起状态下，点击展开按钮 (→)
2. 观察菜单栏恢复完整宽度
3. 观察主内容区域宽度恢复

### 4. 验证状态持久化

1. 收起菜单栏
2. 刷新页面
3. 验证菜单栏保持收起状态

### 5. 菜单导航

1. 点击不同菜单项
2. 验证页面正确切换
3. 验证当前菜单项高亮显示

## 错误处理

| 场景 | 处理方式 |
|------|----------|
| sessionStorage 不可用 | 默认展开状态，功能正常使用 |
| 浏览器不支持 CSS transition | 直接切换，无动画效果 |
